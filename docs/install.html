常见问题

Q：为什么下载 .crx 时 Chrome 直接拦截了？

A：Chrome 75+ 默认禁止从非商店安装 .crx。请改用「右键 → 链接另存为」下载到本地，再按第 3 步拖拽安装。

Q：拖拽 .crx 后提示"无法添加该扩展程序，程序包无效"怎么办？

A：这是因为某些企业版 Chrome 严格限制非商店扩展。请用上方"拖拽被拦截"的兜底方案：改 .zip 解压后「加载已解压的扩展程序」。注意该兜底方式无自动更新。

Q：安装后如何更新到新版本？

A：完全自动。Chrome 每 5 小时检查一次，发现新版会后台下载安装。也可在 chrome://extensions/ 点「更新」按钮立即检查。

Q：如何确认当前版本？

A：打开 chrome://extensions/，找到"JIRA效率助手"，卡片上会显示版本号（如 3.3.0）。

Q：更新后需要重新登录飞书吗？

A：不需要。扩展数据存储在 chrome.storage.local，更新不会清除。飞书登录态会保留。

Q：支持 Edge 浏览器吗？

A：支持。Edge 基于 Chromium，同样打开 edge://extensions/，开开发者模式后拖拽 .crx 即可。
