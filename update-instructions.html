JIRA效率助手 v3.4.2 更新说明

检测到新版本 v3.4.2 可用。请按以下步骤手动更新：

下载 Chrome 安装包 (.crx)
安装步骤：
  1. 点击上方按钮下载 jira-helper-chrome-3.4.2.crx 文件
  2. 打开新标签页，地址栏输入 chrome://extensions 回车
  3. 打开页面右上角的开发者模式开关
  4. 把下载的 .crx 文件拖拽到 chrome://extensions 页面
  5. 弹出确认框点击"添加扩展程序"，完成更新
注意：Chrome 75+ 不再自动更新非 Web Store 的扩展，因此每次有新版本都需要手动下载安装。后续版本会通过扩展内通知提醒您。
