JIRA效率助手 v3.4.4 更新说明

检测到新版本 v3.4.4 可用。请按以下步骤手动更新：

Chrome 更新

下载 Chrome 扩展包 (.zip)
安装步骤：
  1. 点击上方按钮下载 jira-helper-chrome-3.4.4.zip 文件
  2. 把 .zip 文件解压到一个目录(例如 D:\jira-helper 或 ~/jira-helper),路径建议全英文无空格
  3. 打开新标签页，地址栏输入 chrome://extensions 回车
  4. 打开页面右上角的开发者模式开关
  5. 点击页面左上角的"加载已解压的扩展程序"按钮
  6. 选择刚才解压出来的整个文件夹,完成更新
说明：Chrome 130+ 不再允许直接拖拽 .crx 安装非 Web Store 扩展,因此改为下载 zip 解压后"加载已解压的扩展程序"。后续版本会通过扩展内通知提醒您更新。

Firefox 更新

Firefox 用户请按以下步骤更新：

下载 Firefox 安装包 (.xpi)
安装步骤：
  1. 点击上方按钮下载 jira-helper-firefox-3.4.4.xpi 文件
  2. 打开新标签页，地址栏输入 about:addons 回车
  3. 点击页面上方的齿轮图标 ⚙️，选择"从文件安装附加组件"
  4. 选择刚下载的 .xpi 文件，确认安装
  5. 安装完成后重启浏览器即可使用新版本
注意：下载的 xpi 已由 AMO 签名，可直接安装。如果之前通过自定义 update_url 自动更新失败，手动安装一次新版本后即可恢复正常。
